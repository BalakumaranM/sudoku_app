# Fonts Directory

This directory should contain the following font files:

## Required Fonts

### Orbitron (Headings)Orbitron-Black.ttf
Orbitron-Bold.ttf
Orbitron-ExtraBold.ttf
Orbitron-Medium.ttf
Orbitron-Regular.ttf
Orbitron-SemiBold.ttf
- `Orbitron-Regular.ttf` - Download from: https://fonts.google.com/specimen/Orbitron
- `Orbitron-Bold.ttf` - Download from: https://fonts.google.com/specimen/Orbitron

### Rajdhani (Body Text)
- `Rajdhani-Regular.ttf` - Download from: https://fonts.google.com/specimen/Rajdhani
- `Rajdhani-Medium.ttf` - Download from: https://fonts.google.com/specimen/Rajdhani

## How to Download

1. Visit the Google Fonts links above
2. Click "Download family" button
3. Extract the TTF files
4. Copy the required files to this directory

The app will work without these fonts (using system defaults), but for the full "Cosmic Glass" aesthetic, these fonts should be added.
